## Deploy on Vercel

1. **Push this repo to GitHub**
   - Create a new repository on [GitHub](https://github.com/new).
   - Then run:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/bubu-valentines-card.git
   git push -u origin main
   ```

2. **Import on Vercel**
   - Go to [vercel.com](https://vercel.com) and sign in (with GitHub if you like).
   - Click **Add Newâ€¦** â†’ **Project**.
   - Import your `bubu-valentines-card` repo.
   - Leave the defaults (Framework: Vite, Build: `npm run build`, Output: `dist`).
   - Click **Deploy**.

Vercel will build and host the site and give you a URL. Future pushes to `main` will trigger new deployments.
